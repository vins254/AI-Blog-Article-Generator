{% comment %} ============================================================================= FILE: client/templates/index.html PURPOSE: The main application dashboard — the page where users paste a YouTube URL and trigger the full AI article generation pipeline. This is the most complex template in the project. ============================================================================= HOW IT WORKS: 1. The user pastes a YouTube URL into the input field and clicks "Generate Article". 2. A JavaScript fetch() POST request is sent to the /generate-blog endpoint with the URL and the Django CSRF token in the headers. 3. The Django backend returns a Streaming HTTP Response using NDJSON format (Newline Delimited JSON — one JSON object per line). 4. The JavaScript reads the stream chunk by chunk using a ReadableStream reader, updating the progress bar text for each step (1/4, 2/4, 3/4, 4/4). 5. When the final step (step 5) arrives, the full article Markdown is converted to HTML using a lightweight regex parser and injected into the page. 6. Both the URL and the final article are saved to localStorage so the user's work is restored automatically if they refresh the page. FEATURES & TOOLS: - Fetch API + ReadableStream: Enables real-time server-sent data without using WebSockets or SSE. We read the response body incrementally. - NDJSON Streaming: The backend yields one JSON line per stage, and we parse each line as it arrives to update the UI progressively. - CSRF Security: The getCookie() function reads Django's csrftoken from the browser cookie and sends it in the X-CSRFToken request header. - LocalStorage Persistence: URL input is saved on every keystroke (cf_last_url). Generated HTML article is saved after step 5 completes (cf_last_content). Both are restored on DOMContentLoaded, so a page reload brings the user back to exactly where they were. - Lightweight Markdown-to-HTML Parser: A chain of .replace() regex calls converts **bold**, # headings, and - lists from the AI output into real HTML tags before injecting into the DOM. - {% load static %} / {% static %}: Resolves the CSS path correctly in both local development and production environments. ============================================================================= {% endcomment %} {% load static %} ContentFlow — Video Content Automation
